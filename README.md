## 内容

### 机器

- **经济型矿粉制造机**: 前期就可以制作的速度较慢的矿粉制造机，懒狗必备。
- **飞行泡泡**: 可以让周围45格内的玩家获得创造模式飞行能力。

### 物品

- **天使方块**: 可以在脚下放置一个方块，在空中时非常好用。
- **矿物背包**: 一个54格的背包，可以自动收集捡起来的矿物。

还有各种材质的泰拉瑞亚武器。
